> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Give the mobile-only SoroSense app a desktop-native shell + Overview so the demo (run on a laptop, where Freighter lives) stops looking broken — without touching mobile.

**Architecture:** Add a responsive branch to `app/(app)/layout.tsx` (BottomNav on mobile ↔ TopBar + centered max-width container on desktop) and make `home/page.tsx` render a dense desktop Overview at the desktop breakpoint while keeping the existing mobile stack below it. Everything is CSS-breakpoint / small-hook gated; the mobile tree is byte-identical to today.

**Tech Stack:** Next 16 (App Router), React 19, Tailwind v4 (`@theme` in `globals.css`), Vitest + RTL, Playwright.

**Design/pixel source of truth:** `docs/mockups/sorosense-mock-2-desktop.html` (canonical). Spec: `docs/superpowers/specs/2026-07-13-ste-48-desktop-design.md`. When a task says "match the mockup", the exact CSS/markup is read from that file — reproduce its structure, don't invent.

## Global Constraints

- **Mobile unchanged.** Every desktop addition is gated `≥` the desktop breakpoint (`lg:` = 1024px, or a `useIsDesktop()` hook at 1024). The `< lg` tree must match the current DOM/behavior exactly.
- **Invisible safety (R11).** No `Safe`/`Watch`/`risk`/`score`/`Sentinel` string on any surface. Assert explicitly.
- **Segmented is FLAT.** Pressed = `bg-pill text-pill-ink`, idle `text-[#8a8a8a]`, no track, no white raised pill (reuse `components/ui/Segmented.tsx`).
- **Blended-USD always `≈ USD`.** Per-currency buckets never converted. MXN symbol is `$` (`units.ts`).
- **DRY primitives.** Reuse `Card`/`Segmented`/`Switch`/`CoinBadge`/`Button`; do not restyle per screen.
- **U14:** any `fixed` overlay must not sit under a `transform` ancestor (not relevant to Phase 1 — no new overlays here — but the `max-w` container must NOT use `transform` for centering; use `mx-auto`).
- **Gates each task:** `pnpm -r typecheck` (strict `noUncheckedIndexedAccess`) + `pnpm -C frontend lint` + `pnpm -r test`. Playwright at the phase end.

### Task 1: `useIsDesktop` breakpoint hook

**Files:**
- Create: `frontend/hooks/useIsDesktop.ts`
- Test: `frontend/hooks/__tests__/useIsDesktop.test.tsx`

**Interfaces:**
- Produces: `useIsDesktop(): boolean` — `true` when `window.matchMedia('(min-width: 1024px)')` matches. SSR-safe: returns `false` until mounted (mobile-first, matches how `useBuckets` prerenders a loading state; avoids hydration mismatch — KTD7).

- [ ] **Step 1: Write the failing test**
```tsx
import { renderHook } from "@testing-library/react";
import { useIsDesktop } from "../useIsDesktop";

function mockMatchMedia(matches: boolean) {
  window.matchMedia = (q: string) => ({
    matches, media: q, onchange: null,
    addEventListener: () => {}, removeEventListener: () => {},
    addListener: () => {}, removeListener: () => {}, dispatchEvent: () => false,
  }) as unknown as MediaQueryList;
}

test("false before mount, true when the desktop query matches", () => {
  mockMatchMedia(true);
  const { result } = renderHook(() => useIsDesktop());
  expect(result.current).toBe(true);
});

test("false when the query does not match", () => {
  mockMatchMedia(false);
  const { result } = renderHook(() => useIsDesktop());
  expect(result.current).toBe(false);
});
```

- [ ] **Step 2: Run test to verify it fails**
Run: `pnpm -C frontend test useIsDesktop`
Expected: FAIL ("Cannot find module '../useIsDesktop'").

- [ ] **Step 3: Write minimal implementation**
```tsx
"use client";
import { useEffect, useState } from "react";

const QUERY = "(min-width: 1024px)";

export function useIsDesktop(): boolean {
  const [isDesktop, setIsDesktop] = useState(false);
  useEffect(() => {
    const mql = window.matchMedia(QUERY);
    const update = () => setIsDesktop(mql.matches);
    update();
    mql.addEventListener("change", update);
    return () => mql.removeEventListener("change", update);
  }, []);
  return isDesktop;
}
```

- [ ] **Step 4: Run test to verify it passes**
Run: `pnpm -C frontend test useIsDesktop`
Expected: PASS.

- [ ] **Step 5: Commit**
```bash

git commit -m "feat(STE-48): useIsDesktop breakpoint hook (SSR-safe, 1024px)"
```
